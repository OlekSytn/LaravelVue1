Authentication Features:
  • Login
  • Registration
  • Forget-Password Send Recover Link Mail
  • Reset Password
Project Features
  • Category CRUD API
  • Product CRUD API with category & user relationship

Base URL

http://localhost/Learning/appnap-assignment/backend/api/v1

Authenticating requests

This API is not authenticated.

Authentication

Login

Example request:
const url = new URL(
    "http://localhost/Learning/appnap-assignment/backend/api/v1/api/v1/auth/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "mdrabiulhasan.me@gmail.com",
    "password": "123456"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
